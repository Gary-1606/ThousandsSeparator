Thousand Separator Library

The base function takes the input as numbers or array of numbers and returns the numbers formatted with separated thousands (adding "," at appropriate places))

Examples: How to access

conversionlib.thousandSeparator(5318008); // "5,318,008"
conversionlib.thousandSeparator([9876543.21, 100000]); // ["9,876,543.21", "1,00,000"]
conversionlib.thousandSeparator(9876543.21); // "9,876,543.21"
